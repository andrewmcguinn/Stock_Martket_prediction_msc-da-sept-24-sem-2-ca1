# Stock_Martket_prediction_msc-da-sept-24-sem-2-ca1


**Predicting Stock Prices using Hadoop, Apache Spark, and Deep Learning (RNN & LSTM)**

A large-scale stock market prediction project completed as part of my **MSc in Data Analytics**, integrating **distributed computing** and **deep learning** techniques.
This project demonstrates the ability to **process large historical datasets**, **apply advanced sequential models (RNN & LSTM)**, and **forecast stock prices** with improved accuracy compared to traditional machine learning models.

---

## üìå Overview

Stock prediction is a **critical yet challenging task** due to the inherent volatility and complexity of financial markets.

This project integrates **Hadoop**, **Apache Spark**, and **deep learning models** to predict stock prices using historical data. Key contributions include:

* Hadoop for **distributed data storage**
* Apache Spark for **large-scale preprocessing** and **distributed computation**
* Deep learning using TensorFlow, including **Recurrent Neural Networks (RNNs)** and **Long Short-Term Memory (LSTM) networks**, capturing temporal dependencies in stock prices

The study analyzes over **10 years of historical stock data**, automatically downloaded using Python libraries for forecasting purposes.

## üìä Methodology

### Data Acquisition

* Historical stock price data is automatically downloaded using Python libraries
* Over **10 years of stock records** are collected for model training

### Distributed Processing

* **Hadoop** manages scalable storage of raw data
* **Apache Spark** preprocesses data efficiently, enabling parallel computations and handling large datasets

### Deep Learning Models

* **Recurrent Neural Networks (RNNs)** for sequential pattern recognition
* **Long Short-Term Memory (LSTM)** networks to capture long-term dependencies in stock price sequences
* Models trained and evaluated using TensorFlow

### Model Evaluation

* Predictions are compared to actual stock prices
* Performance metrics assess the superiority of deep learning models over traditional machine learning models

---

## üìà Key Insights

* LSTM networks **outperform RNNs** and traditional ML models for stock price forecasting
* Distributed computing with **Hadoop and Spark** enables efficient handling of large-scale financial datasets
* Automated data acquisition allows reproducible and scalable forecasting experiments
